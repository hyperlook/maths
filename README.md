# Blackboard Long Screenshot Print Assistant (黑板长图打印助手)

## 简介
这是一个专为暗黑背景、彩色手写字体（如教学黑板演示截图）设计的打印辅助工具。能够智能反色、保护白色底图、并完美适配 A4 纸张进行分页打印。

## 主要功能
- **智能反色调优**：自动识别黑板背景并进行反色处理，过滤浅灰底色，既省墨又美观。
- **智能区域保护**：自动识别并保护截图中原本就是“白底黑字”的试卷或文档框，防止其被错误反色。
- **A4 自动分页**：根据 A4 纸张比例自动对长截图进行切分，支持打印预览。
- **极致移动端体验**：专门适配 iPhone，支持流畅的滑块实时切口调节，所见即所得。

## 食用方法
1. 点击或拖拽一张黑板长截图到页面中。
2. 拖动手中的滑块调节：
   - **背景白位**：控制背景的纯净度。
   - **文字增强**：增强字体的深浅对比。
   - **内容边距**：预留装订或打孔位置。
3. 如果自动分页点切断了重要文字，使用每页下方的**滑块调整切口**。
4. 点击“预览并开启打印” -> 选择“另存为 PDF”或直接连接打印机。
